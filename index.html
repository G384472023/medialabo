各授業回の学習事項

第1回 WWW と HTML

URLとはプロトコル、サーバー、パスの3つで構成されている。 Webページを構成する部分はHTML、CSS、JavaScipt、画像・音声・動画の大きく4つに分けられている。 HTMLにはブロック要素とインライン要素があり、段落（p要素）や見出し文字、メニューバーなどはブロック要素で、 強調（strong要素）やリンクなどはインライン要素です。 また、インライン要素の中にブロック要素は配置することができない。

演習の成果: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

ページを装飾する記述はCSSと言う。 CSSをHTMLに適用する方法は外部スタイルシート、内部スタイルシート、インラインスタイルの3つの方法がある。 CSSの書式には装飾したい要素を指定するセレクタ、装飾の種類を指定するプロパティ名、装飾の具体的な値を指定するプロパティ値がある。

演習の成果: ウェブブラウザの紹介

第3回 JavaScipt の基本

JavaScriptとJavaは、変数宣言や比較、型変換、文字列、関数の構文が違う。 関数宣言の外で宣言するを グローバル変数 と言い、ページを開いている間、ずっと使用できる。 for-of構文とは配列の要素1つずつに対して処理を実行する。順番はどうでもよくて、配列要素それぞれに対して、繰り返し何かしたいというときに使う

演習の成果: オブジェクトの参照

第4回 DOM と JavaScript

DOMは Web コンテンツを表現するデータ構造。DOM を使用することによって、JavaScript などのプログラムによって、Web コンテンツの内容を読み取ったり、コンテンツに変更を加えたりする処理が書けるようになる。 要素の検索をする構文は主に３つある。 1つ目は、document.querySelector()。HTML の中から引数に指定された要素を1つ検索し、引数には セレクタ を書く。 2つ目は、document.querySelectorAll()。セレクタに指定されたすべての要素を検索する。 3つ目は、querySelectorAll()。引数に指定するセレクタを p とすると、HTML 全体で使用される全ての p 要素を検索する。

演習の成果: DOM プログラミング演習

第5回 イベント処理

イベントはWebページ上で起こった動作や出来事のことです。例えば、 ユーザーがある要素の上をマウスでクリックしたり，ある要素の上にカーソルを持ってくる ユーザーがキーボードのキーを押す ユーザーがブラウザー画面をリサイズしたり閉じたりする イベントが発生した時に実行する処理のことを イベントハンドラ と言います．イベントハンドラは関数として定義します． label 要素は入力項目の説明する働きを持ちます． ページ上では単なるテキストとして表示されます for 属性は説明する入力項目を意味します input 要素は入力項目を意味します type 属性は入力する値の種類です。type="text" によって，ページにはテキストボックスが表示されます name 属性はこの input 要素の名前 size 属性はテキストボックスの横幅

演習の成果: イベントの種類
