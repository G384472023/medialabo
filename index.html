各授業回の学習事項

第1回 WWW と HTML

URLとはプロトコル、サーバー、パスの3つで構成されている。 Webページを構成する部分はHTML、CSS、JavaScipt、画像・音声・動画の大きく4つに分けられている。 HTMLにはブロック要素とインライン要素があり、段落（p要素）や見出し文字、メニューバーなどはブロック要素で、 強調（strong要素）やリンクなどはインライン要素です。 また、インライン要素の中にブロック要素は配置することができない。

演習の成果: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

ページを装飾する記述はCSSと言う。 CSSをHTMLに適用する方法は外部スタイルシート、内部スタイルシート、インラインスタイルの3つの方法がある。 CSSの書式には装飾したい要素を指定するセレクタ、装飾の種類を指定するプロパティ名、装飾の具体的な値を指定するプロパティ値がある。

演習の成果: ウェブブラウザの紹介

第3回 JavaScipt の基本

JavaScriptとJavaは、変数宣言や比較、型変換、文字列、関数の構文が違う。 関数宣言の外で宣言することをグローバル変数と言い、ページを開いている間、使用できる。 for-of構文とは配列の要素1つずつに対して処理を実行する。順番は関係なく、配列要素それぞれに対して、繰り返し何かしたいというときに使う。

演習の成果: オブジェクトの参照

第4回 DOM と JavaScript

DOMは Web コンテンツを表現するデータ構造。DOM を使用することによって、JavaScript などのプログラムによって、Web コンテンツの内容を読み取ったり、コンテンツに変更を加えたりする処理が書けるようになる。 要素の検索をする構文は主に３つある。 1つ目は、document.querySelector()。HTML の中から引数に指定された要素を1つ検索し、引数には セレクタ を書く。 2つ目は、document.querySelectorAll()。セレクタに指定されたすべての要素を検索する。 3つ目は、querySelectorAll()。引数に指定するセレクタを p とすると、HTML 全体で使用される全ての p 要素を検索する。

演習の成果: DOM プログラミング演習

第5回 イベント処理

イベントは、Webページ上での動作や出来事を指します。 例として、ユーザーが要素をクリック、カーソルを合わせる、キーボードのキーを押す、ブラウザー画面をリサイズまたは閉じるといった操作があります。 これらのイベントが発生した際に実行される処理をイベントハンドラと呼び、関数として定義されます。 label要素は入力項目を説明するために使われ、ページ上ではテキストとして表示されます。 for属性は対応する入力項目を示します。input要素は実際の入力項目を表し、type属性で入力の種類を指定します。 例えば、type="text"はテキストボックスを表示します。name属性はinput要素の名前を指定し、size属性はテキストボックスの横幅を設定します。

演習の成果: イベントの種類

第6回 プロジェクト課題を完成させよう

AJAXは、ページ全体を再読み込みせずに一部を更新する処理です。 これはJavaScriptのXMLHttpRequestオブジェクトを用いて通信し、HTML、CSS、JavaScript、DOM、イベント処理を組み合わせて行います。 axiosはAJAX通信を簡潔に実現するライブラリで、XMLHttpRequestを内部で使用しつつ、より分かりやすくプログラムを記述できます。 通信はブラウザからサーバへのデータ要求（リクエスト）とサーバからブラウザへのデータ返答（レスポンス）に分かれ、インターネット経由で行われます。 リクエストからレスポンスまで時間がかかるため、通信処理には同期通信（レスポンスを待つ）と非同期通信（待たずに別の処理を実行し、レスポンスはイベント処理として受け取る）の2方式があります。
